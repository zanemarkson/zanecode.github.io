NEqMD-ToolKit
This toolkit is a C/Fortran based framework originally written for assisting theoretical/computational chemistry research where Non-Equilibrium MD simulation is needed. It provides a set of command-line tools as well as an easy-to-use C style library (libNEqMD). Functions included in the libNEqMD not only performs a wide range of basic mathematical operations, but also enables one to parse and edit text files. The ultimate goal of this on-going project is to provide a framework and toolkit so that users can perform complex mathematical operations and file operations directly from Unix/Linux shell instead of having to invoking a programming environment such as MATLAB. All tools in this toolkit are written in a single-thread fashion. Therefore, in order to enable the parallelization, an MPI wrapper is included. Detailed description of this project can be found https://github.com/zanemarkson/NEqMD.ToolKit.

Interactive Drawing Board
The aim of this project is to build an application that enables people to have a remote shared drawing board. It was inspired by the idea to have a virtual whiteboard so that during video conferences participants can collaborate by simply drawing or writing down the ideas. This project is mainly built using javascript stack in web technology (MEAN stack), including node.js and express.js for the backend server and routing, socket.io for the websocket communication between front and back end, angular.js and bootstrap 3 for building the UI and also mongoDB for the database service. The Electron framework, because of its cross-platform versatility, is used to pack the front and back end into stand-alone desktop applications. Currently the web-based application and a desktop app for Mac OS have been prepared. The code and description can be found at https://github.com/zanemarkson/DrawingBoardApp.

Remote Control for Reveal.js
This project is a fork of the "chalkboard" plugin for reveal.js presentation framework. It is built using similar javascript technologies as in the interactive drawing board project. This plugin enables one to control and make annotations on the presentation using mobile devices and potentially enables multiple audiences to participate into a remotely hosted presentation.
